This code is build by Visual Studio 2019 with libraries below installed.
-openCV 4.2.0
-fltk-1.3.5
-freeglut 3.0.0 MSVC Package (https://www.transmissionzero.co.uk/software/freeglut-devel/)

This source code is the revised version of the CAD18 paper:
Yuka Watanabe, Jun Mitani, "Interactive Modelling of Curved Folds with Multiple Creases Considering Folding Motions", Computer-Aided Design & Applications, 16(3), 2019, 452-465.

The operation verification have not been done for all functions after some minor rivisions.
Bugs will be fixed as soon as I find them.
